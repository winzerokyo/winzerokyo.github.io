SWITCH模拟器的问题汇总
发表于：2025-02-01 | 分类： SWITCH

模拟器闪退的自我排查，闪退没有固定的解决方法，因为产生原因多种多样。可以按照下面的办法自己排除。闪退靠排除，1看看自己的显卡驱动够不够新，这个去官网下载最新就能解决。2将MOD和金手指全部不用，排除这些东西导致的闪退。3加载了别人的着色器缓存，这玩意不是不好而是每个硬件生成的会有不一样而且硬件间会有不兼容的情况，正常删除着色器缓存重新生成就可以。4软件冲突，这些体现在桌面花里胡哨（桌面美化）和喜欢游戏帧率监测（用游戏++或者其他游戏帧率监测的）或者录屏或者用推流的人，因为这些软件会使模拟器的图形接口无法正常调用导致闪退。5设置过于激进，这个就是对模拟器的人没什么认识的人犯的错误，将模拟器设置调得过高，导致机器无法处理闪退。6插好各种音响设备，包括耳机或者其他能输出声音的设备，要能够被系统识别的输出声音的设备。

整合好的游戏要慎用，有可能会进不了游戏，进不了游戏就找游戏本体，游戏更新和DLC分开加载的资源来就可以，龙神模拟器那里右击游戏图标，管理游戏更新和管理游戏DLC那里能不能识别加载，不能就自己去网上下载游戏更新和DLC来加载

问题1：游戏无法打开，打开后闪退或者黑屏。
答案：
情况1将模拟器升级或者降级处理，因为某些版本的模拟器会出现这样的问题，该问题出现原因
原因1模拟器与游戏或者电脑环境不兼容，升级或者降级模拟器（up主本人，B站网友喔嘛麻，B站网友三长，B站网友救济魔女まどか，B站Hestia-L）
原因2
Yuzu模拟器出现原因
情况1更新显卡驱动
情况2卸载/更新有问题的屏幕录制（覆盖屏幕）或帧数监控软件
情况3“模拟”-“设置”-“图形”-单击“API”选择vulkan，看看有没有显卡型号，验证您的Vulkan安装，如果在vulkan下没有显示显卡型号可以试下以下两种方法
（1）API选项调成OpenGL再改成vulkan
（2）重启模拟器将API选项调成OpenGL试一下能否运行游戏，之后将OpenGL改成vulkan
情况4有可能是内存频率调节比较激进导致。

龙神模拟器出现原因
情况1更新显卡驱动
情况2卸载/更新有问题的屏幕录制（覆盖屏幕）或帧数监控软件
情况3“options”-“settings”-“graphics”-单击“graphics backend”选择vulkan，看看“preferred gpu”选项有无显卡型号，验证您的Vulkan安装，如果在vulkan下没有显示显卡型号可以试下以下两种方法
（1）“graphics backend”选项调成OpenGL再改成vulkan
（2）重启模拟器将“graphics backend”选项调成OpenGL试一下能否运行游戏，之后将OpenGL改成vulkan
（B站网友猎神邪天、B站网友暴走弗洛鸽，B站网友第五期团下等马，B站网友愛丽丝的囚徒）；】
情况2游戏本体有问题（B站网友斑点丶丶，B站网友抽象鲨）；
情况3内存少于16GB有几率因为运行内存不足而无法打开游戏；
情况4电脑运行着游戏++等软件导致图形接口被占用，导致无法识别。

问题2：柚子模拟器的固件和密钥和龙神模拟器视频的固件密钥是一样的吗？
答案：固件和密钥两个模拟器是通用的。（B站网友石炎）

问题3：画音不同步
答案：这个有可能是因为模拟帧率不够导致。Yuzu可以将cpu的精度设置为精确有几率解决（B站网友迷糊小仙客、UP主本人）

问题4：手柄无法使用
答案：手柄设置里面所有或者部分按键出unbound字样，只要重新在手柄设置处选好输入设备，然后点击unbound，输入你设置按键（B站网友烙印の光）

问题5：龙神无法加载游戏文件夹（出自第三方编译的龙神模拟器）
答案：
情况1应该是模拟器本身出了错误，选用官方最新版本升级第三方编译的龙神模拟器可能会有改善。（B站网友火山动物园）
情况2游戏本体有问题

问题6：纯键盘要如何操作
答案：手柄设置里面选好输入设备为keyboard，然后点击相应按键进行设置，输入你设置按键（B站网友梅川内酷晓解）

问题7：某些游戏运行时候有卡顿或者声音和画面不同步的问题（例如暗黑破坏神3，GTA3 最终版，Sympathy Kiss）
答案：这个问题主要出现在yuzu系模拟器声音设置，输出设备改成sdl2就能解决（up主本人）

问题8：龙神感觉不小心开了两倍速，就是关闭VSync如何解决
答案：在游戏时将左下角的VSync点击一下变成非红色字状态就能解决。（B站皮皮鸿鸿）

问题9：玩游戏时画面发黄
答案：主要是可能是因为模拟器的问题或者是使用AMD显卡，通常是升级模拟器可以解决。（百度ryujinx贴吧）

问题10：模拟器无法打开，或者无法打开设置
答案：
情况1龙神模拟器ryujinx文件夹没有放在默认位置C:\Users\用户名\AppData\Roaming（仅限没在模拟器文件夹下建立portable文件夹的用户。）（B站网友三原怜）
情况2系统必须是Windows10 1803及以上的版本才能使用，如果达不到系统要求就会弹出错误框提示升级系统（B站网友烈火炜哥）
情况3yuzu系模拟器将C:\Users\用户名\AppData\Roaming\yuzu\config下的qt-config.ini文件删除然后重启模拟器（官方FAQ）
情况4柚子模拟器一些屏幕录制应用程序使用可疑的方法将自己注入软件，从而导致崩溃。例如Reshade、GShade、Bandicam、Action和Screenrec或者微星小飞机。如果已安装软件，请将其卸载。对于Reshade GShade，如果您需要至少安装它，请关闭它在窗口上全局触摸Vulkan的功能，以允许Yuzu&Vulkan再次工作。官方建议使用OBS Studio、Radeon ReLive、Nvidia ShadowPlay或Microsoft XBox Game Bar。（官方FAQ）
情况5VC++2015-2022版本不够新，更新即可以

问题11：手柄控制失灵，某些控制失控
答案：
情况1将player1以外的其他player（包括handheld）设置的输入设备为disable即可解决。（B站网友球灬球Sama）
情况2手柄设置中Deadzone Left或Deadzone Right下面的数值为0，导致手柄过度灵敏失控，将数值跳到0.10或以上可以改善。（B站网友丹墀之下）

问题12：ryujinx.exe-无法找到入口
答案：把publish\bin里的zlib1.dll文件，复制到Ryujinx.exe文件的同目录下。（ryujinx吧）

问题13：Ryujinx.Ava已停止工作
答案：ryujinx文件夹没有放在默认位置C:\Users\用户名\AppData\Roaming（仅限没在模拟器文件夹下建立portable文件夹的用户。）（B站网友三原怜）

问题14：龙神模拟器宝可梦朱紫运行时闪退
答案：一般情况下系统设置上Memory Manager Mode：选Host（fast），（1.1.366之前）Expand DRAM Size to6GiB不选就可以（B站网友周大刀xxx）

问题15：未知软件异常（0xe0434352），有一个错误框写着 you are running an outdated version of windows.
答案：升级系统，系统必须是Windows10 1803及以上的版本才能使用，如果达不到系统要求就会弹出错误框提示升级系统（B站网友烈火炜哥）

问题16：龙神连接不上joy双手柄
答案：龙神模拟器本身识别不了joy双手柄，只能单独识别joy左手柄或右手柄，需要betterjoy驱动来映射，建议使用pro手柄（B站网友藤原书记爱跳舞）

问题17：龙神模拟器将控制台隐藏
答案：将options-show log console这个不选就可以

问题18：yuzu开启时出现错误“VCRUNTIME140_1.dll was not found”
答案：安装64位版本VC++2015，2017，2019，2022（官方FAQ）

问题19：yuzu开启时出现错误“Broken Vulkan Installation Detected”
答案：
情况1更新显卡驱动
情况2卸载/更新有问题的屏幕录制（覆盖屏幕）或帧数监控软件
情况3“模拟”-“设置”-“图形”-单击API选择vulkan，看看有没有显卡型号，验证您的Vulkan安装（官方FAQ）

问题20：yuzu的基础电脑配置（用龙神的可以参考一下）
答案：
win10 1803版本以上
支持OpenGL4.6的显卡
8G物理内存
高性能单核处理器（官方FAQ）

问题21：yuzu开启时出现错误“Missing Derivation Components”
答案：将 prod.keys和title.keys放在C:\Users\用户名\AppData\Roaming\yuzu\keys位置（仅限没在模拟器文件夹下建立portable文件夹的用户。）（官方FAQ）

问题22：yuzu开启时出现错误“Unable to start application: Os { code: 2, kind: NotFound, message: “The system cannot find the file specified.” }”
答案：杀毒软件误杀模拟器为病毒（官方FAQ）

问题23：yuzu游戏启动失败弹出“WerFault.exe - Application Error - The application was unable to start correctly”
答案：物理内存不足，如果物理内存足够可能需要启用“模拟”-“设置”-“通用”-“扩展的内存布局（6GB DRAM）”（官方FAQ）

问题24：游戏存档无故损坏无法读取存档
解决思路将存档位置的文件备份后逐个相关文件删除验证，当某个有问题的文件删除后存档能读取就说明该存档的这个文件损坏导致存档无法读取，这种方法可以保存大部分存档数据和进度。

问题25：ryujinx（龙神）模拟器进游戏的时候是透明的，游戏背景变透明，看到桌面背景
答案：关闭显卡驱动里面的全局锐化功能

问题26：龙神如何制作MII（某些游戏可能需要用到,例如迷托邦）
答案：在File-Load applet-MII Editor根据提示操作制作即可，再在游戏中选择即可，yuzu系可以在游戏中建立不需要额外操作或者工具-打开MII editor来生成

问题27：真女神转生5不显示中文
答案：模拟器系统设置上将语言设置成繁体中文才能正常显示中文

问题28：某系游戏无法正常运行（迪士尼梦幻星谷）
答案：游戏一定要加载补丁

问题29：某些游戏本体没有中文（塞尔达传说旷野之息，剪纸大厦，妖怪捉迷藏，马里奥赛车8）
答案：加载最新升级补丁就有中文

问题30：暗黑破坏神3整合或者加载中文DLC无中文
答案：DLC中的中文语言包需要在加载所有补丁和DLC（除语言包DLC）后再加载才能生效，龙神无法提供中文语音，有中文显示

问题31：龙神模拟器设置手柄体感模式
答案：这个手柄必须要有陀螺仪和支持体感才能用，支持switch模式。
首先手柄切换成switch模式，链接电脑后模拟器可以正确是识别。左上角就会显示设备可以选择，然后再勾选体感模式motion第一个选项即可，感谢提供答案的B站网友ziopp99

问题32：AMD有核显的处理器和Nvdia独显搭配无法启用Vulkan模式，OpenGL很卡的解决方法
答案：开始菜单搜索编辑系统变量，打开的页面最底下有环境变量，点击，打开的页面找到系统变量，点击系统变量下方的新建，变量名为
DISABLE_LAYER_AMD_SWITCHABLE_GRAPHICS_1 变量值为1，确定就行，不行话设置完重启电脑
原理应该是因为AMD核显和Nvdia独显的驱动相互冲突，透过屏蔽核显来正常释放独显性能（解决方法源于贴吧Shark7帖子）

问题33：输入设置中使用键盘作为输入设备，但是游戏中无法使用
答案：首先看看输入设置中按键有没有设置好，然后启动模拟器后要将输入法调成英文输入法，模拟器使用键盘作为输入设备不支持中文输入法

问题34：柚子模拟器和龙神模拟器如何安装update和DLC？
答案：柚子模拟器选择“文件”-“安装文件到NAND”然后选择所有的update和DLC安装即可。龙神模拟器右击游戏图标选择Manage Title Updates将所有update文件添加后选择要用补丁版本即可，右击游戏图标选择Manage DLC将所有DLC文件添加即可。

问题35：龙神模拟器无法运行游戏，控制台出现Gtk-CRITICAL**：14：49：30.035类似语句
答案：将模拟器最大化窗口就可以解决。

问题36：宝可梦剑盾在冰系道馆卡死闪退
答案：这个是区分剑和盾游戏的道馆，可能是剑和盾游戏存档互相调换了。如果是在剑遇到就将存档转移到盾那里，另外一个也是调换存档来操作。这种情况多出现在网上下载别人存档时发生，请别乱用网上别人的存档。

问题37：A卡Vulkan模式不显示显卡型号
答案：如果是更新到最新驱动还是不显示就将驱动降到最新前一版的驱动

问题38：更新（update）文件或者DLC文件安装失败
答案：根据百度贴吧埃阿斯的方法，使用sak软件的path nsp功能将无法安装的文件放进去修复，修复后的文件就应该能使用了。原理据本人估计应该是在在机子搬出来时游戏的内部路径出了错误，用这个功能将路径修复了，文件就能正常使用了。

问题39：月姬REMAKE V1.0.2+CHS V2.02和夜点灯（深夜点灯），使用汉化MOD后画面出现错误
答案：经大概测试应该是模拟器编译优化王国之泪时导致的。因为这是在1.1.811以后出现的，这个时候两个模拟器都针对王国之泪进行大量的优化。暂时只知道龙神1.1.811柚子主线版1411之后的版本都有这个问题，那段时间的柚子和龙神都一样有这个画面错误。解决方法要么模拟器重新支持，要么汉化MOD那里进行修正。（月姬有官方中文的版本了可以不用这个方法了，这个方法是用在美版的）

问题40：柚子宝可梦朱和紫在宝可梦进化时无法正常运行，宠物进化时卡死，可以拖动视角，也有声音
答案1：可以在进化前保存，关闭多核仿真功能，然后进化后保存游戏后，开启多核仿真功能调回来
答案2：贴吧有吧友说卸载服装DLC也可以解决这个问题

问题41：柚子固件更新后无法读取存档
答案：用更新工具更新的，换KEY的时候这个文件不要删title.keys_autogenerated这个文件，删了就读不到原来的存档，从回收站找回来的。放回原位置就好了。

问题42：洛克人EXE传奇合集1和2的运行问题
答案：洛克人EXE传奇合集1 V1.0.3+5DLC主线版柚子只能运行游戏本体，不能使用update和DLC。龙神如果要加载全部DLC就不能加载update。加载了update，DLC的Mega Man Battle Network Legacy Collection Vol. 1 [P.Aデビューチップセット② Vol.1] [010038E016265005][v0].nsp就不能加载，否则会闪退
洛克人EXE传奇合集2 V1.0.3+5DLC主线版柚子无法运行，龙神如果要加载全部DLC就不能加载update。加载了update，DLC的Mega Man Battle Network Legacy Collection Vol. 2 [P.Aデビューチップセット② Vol.2] [0100734016267005][v0].nsp就不能加载，否则会闪退

问题43：龙神模拟器无法识别宝可梦朱紫的保存数据（存档数据）
答案：在Ryujinx文件夹里找到poke-trade文件删除就行了。（B站网友“Hui钛狼哦”提供的解决方法）

问题44：超级马里奥派对 V1.0.1运行游戏时进入游戏会闪退
答案：将VULKAN改成OpenGL就能运行，还有要注意这个游戏要用joycon玩才能完整体验。

问题45：游戏运行时人物视角或者人物移动不断旋转？
答案：柚子上可能是因为在输入设置中使用了“鼠标漂移”的设置，只要不启用就可以解决。

问题46：星之海洋2 第二个故事R V1.0.1+4DLC运行时声音播放不正常
答案：声音插件使用SDL2就可以正常播放声音。

问题47：超级马里奥3D全明星 V1.1.1怎样运行在模拟器上？
答案：这个游戏是多游戏集合在一个文件上，建议使用龙神模拟器运行游戏，图形接口选OpenGL，我用的是1.1.1065的官版龙神玩的。系统固件16.1.0。这样运行时卡顿会比较少，但是这游戏对处理器单核要求比较高，本人估计可能是游戏里套了个模拟器那类的软件来运行里面的游戏。

问题48：东方计划 不可思议的幻想乡 重载 V1.0.1（不可思议的幻想乡tod reload）播放开场动画后黑屏无法继续游玩
答案：只能在动画播放时直接跳过，跳过了就能玩了

问题49：寒蝉鸣泣之时 奉用龙神模拟器有破音的问题
答案：换yuzu来玩。本人使用龙神1.1.1202版本时经过设置都没法解决。

问题50：笔记本电脑无法跑满频率导致游戏帧率低下
答案：以下是Nvdia的显卡的解决方法，去Nvdia GeForce Experience那里去常规-WHISPER MODE这个选项是不是打开了，如果是打开了，关闭了就可以跑满性能了。（贴吧迷茫浪子提供的解决方法）

问题51：精灵宝可梦 去吧,伊布! 和精灵宝可梦 去吧,皮卡丘! 设置按键后没法使用
答案：这两个游戏在模拟器中只支持左joycon或者右joycon手柄类型，不是这两个手柄类型的的按键在游戏中是无效的。

问题52：病态 七侍者 V1.0.6运行在PC版模拟器运行方法
答案：1这个游戏要用柚子系的模拟器才能玩，龙神的玩不了。
2柚子的主线版1732或者EA版的4176只能使用图形接口中的OpenGL接口，vulkan用不了。运行中有图像问题，暂停等某些画面有图像错误。
sudachi、suyu或者torzu的最新版可以用vulkan接口，而且没有图像问题
3每次重新运行游戏前要删除游戏所有的着色器缓存，不然进不了游戏
4如果前三步做了还是进不了游戏请关闭模拟器重新启动模拟器，记得运行游戏做步骤3就可以。

问题53：真女神转生5 复仇 V1.01+7DLC改名字后闪退
答案：此方法暂时适用于PC版的模拟器上
使用sudachi V1.0.2版本，CPU设置那里使用“unsafe”选项选后的选项不用管，GPU要使用“OpenGL”，由于本人是用N卡的，用的是OpenGL中N卡专用的api，A卡你要自己试试A卡专用的。

问题54：龙神模拟器在添加游戏文件夹位置时闪退
答案：1有可能是输入法跟模拟器不兼容导致的闪退，现在有实例表明，搜狗的中文输入法会跟模拟器有冲突，可以转系统自带输入法解决。
2关闭挂梯子软件（B站网友穗绀提供的答案）
3打开portable文件夹里的config文件，game_dir里面的内容清空（B站网友穗绀提供的答案）

问题55：龙神模拟器祖传VULKAN模式下无法识别显卡型号
答案：1更新好显卡驱动，驱动版本过低会造成无法识别
2删除游戏PP或者叫游戏++，微星小飞机等帧率监测软件，这些软件会有几率造成显卡接口无法识别的问题。
最后做了上述步骤后将用户文件夹下的Config.json删了，重启模拟器试试，上述只是相关可能性，不一定是适合正确解决方法。

问题56：新版本龙神无法在超级马里奥 奥德赛中调用中文
答案：请将系统语言中的改成简体（中文）–无效的那个选项，选了就能调用该游戏的中文选项。

问题57：疯狂兔子 奇遇派对无法控制？
答案：输入设置上要使用左或者右的joycon类型手柄来设置按键。非原生joycon手柄，摇杆要设置顺时针旋转90度，然后左joycon中，方向下键设置成A键，方向左键设置成B键，方向上键设置成Y键，方向右键设置成X键。然后右joycon中，X键设置成A键，A键设置成B键，B键设置成Y键，Y键设置成X键，这样就可以正常游玩了。

问题58：安卓版的NS模拟器在某些要自己输入文字的地方闪退
答案：
1.进入用户文件夹nand\system\Contents\registered
删除固件文件5b991be1efff3e2d2bc34fba9d200c58.nca就能调用设备的输入法
2.直接不安装固件
